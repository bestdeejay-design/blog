## 🎯 Краткое содержание (TL;DR)

**Что это:** Система управления новостями для нескольких сайтов.

**Зачем:** Одна админка для всех сайтов, публикация в несколько каналов сразу.

**Кто может:** Супер-админ (всё), Админ (каналы), Редактор (новости).

**Как начать:** 
1. Создать Supabase проект (10 мин)
2. Выполнить SQL скрипт (2 мин)
3. Заполнить .env.local (5 мин)
4. Запустить `npm run dev` (1 мин)
5. Стать супер-админом (3 мин)

**Как интегрировать:** 
```tsx
import NewsFeed from '@/components/news-feed'
<NewsFeed baseUrl="https://your-app.com" limit={5} />
```

**Сколько стоит:** $0 (бесплатные тарифы Supabase + Vercel)
