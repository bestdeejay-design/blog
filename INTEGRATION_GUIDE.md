# 📰 Интеграция новостного виджета в сайт АТМОСФЕРА (arental)

## 🎯 Что нужно сделать:

### Шаг 1: Вставить стили и скрипт виджета

В файл `index.html` сайта **arental**, найти секцию где хотите показать новости (например, после блока с услугами).

**Перед закрывающим тегом `</section>`** вставить содержимое из файла:
```
news-widget-embed.html
```

Это добавит:
- ✅ Стили для сетки новостей
- ✅ HTML контейнер
- ✅ JavaScript для загрузки новостей (до 12 штук)

---

### Шаг 2: Добавить модальное окно

**Перед закрывающим тегом `</body>`** вставить содержимое из файла:
```
news-modal-embed.html
```

Это добавит:
- ✅ Модальное окно для просмотра новости
- ✅ Стили модалки
- ✅ Скрипт открытия/закрытия

---

## 📍 Пример куда вставить:

Найти в `index.html` секцию (примерно строка 327):

```html
</section>

<section id="channels" class="section">
```

**Между ними** вставить:

```html
<!-- NEWS WIDGET START -->
<section class="section news-widget-section">
  <div class="container">
    <span class="eyebrow">Новости</span>
    <h2 class="h2">Последние обновления</h2>
    <p class="kicker">Будьте в курсе наших достижений и инноваций</p>
    
    <!-- Сюда вставить код из news-widget-embed.html -->
  </div>
</section>
<!-- NEWS WIDGET END -->
```

---

## ⚙️ Настройки:

### Изменить количество новостей:
В `news-widget-embed.html` найти строку:
```javascript
const limit = 12;
```
Поменять на нужное число.

### Изменить API URL:
Если нужно использовать другой адрес:
```javascript
const API_URL = 'https://your-domain.com/api/news-simple';
```

## 🔧 Если что-то не работает:

1. **Проверить CORS** - API должен разрешать запросы с вашего домена
2. **Консоль браузера** - посмотреть ошибки (F12 → Console)
3. **Проверить API** - открыть https://blog-three-opal-85.vercel.app/api/news-simple?limit=12
