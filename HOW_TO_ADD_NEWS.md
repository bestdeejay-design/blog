# Как добавить новость

## Шаг 1: Настройка базы данных

1. Откройте [Supabase Dashboard](https://supabase.com/dashboard)
2. Выберите ваш проект
3. Перейдите в **SQL Editor**
4. Скопируйте и выполните содержимое файла `supabase/migrations/20260321_create_database_schema.sql`

Это создаст все необходимые таблицы:
- `channels` - каналы (Telegram, YouTube, VK и т.д.)
- `user_profiles` - профили пользователей
- `news` - новости
- `news_channels` - связь новостей с каналами

## Шаг 2: Запуск приложения

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Шаг 3: Вход в админ-панель

1. Откройте http://localhost:3000/login
2. Войдите под учетной записью супер-админа:
   - Email: `admin@blog.local`
   - Пароль: `Admin@2026Secure!`

## Шаг 4: Добавление новости

### Способ 1: Через админ-панель (рекомендуется)

1. После входа откроется панель управления
2. Убедитесь, что выбрана вкладка **"📰 Новости"**
3. Нажмите кнопку **"➕ Добавить новость"**
4. Заполните форму:
   - **Заголовок** - название новости (обязательно)
   - **Текст новости** - содержимое (обязательно)
   - **Канал** - выберите канал публикации (обязательно)
   - **Статус** - черновик или опубликовано
5. Нажмите **"✅ Создать новость"**

### Способ 2: Через API

```bash
curl -X POST http://localhost:3000/api/news/create \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "title=Моя новость" \
  -d "content=Текст новости" \
  -d "channel_id=UUID_КАНАЛА" \
  -d "status=published"
```

## Шаг 5: Создание канала (если нет каналов)

Перед добавлением новости нужно создать хотя бы один канал:

1. В админ-панели перейдите на вкладку **"📺 Каналы"**
2. Нажмите **"➕ Добавить канал"**
3. Заполните:
   - **Название** - например, "Main News"
   - **Ключ (slug)** - например, "main-news" (только латиница, цифры и дефис)
4. Нажмите **"✅ Создать канал"**

## Проверка результата

После создания новости:
- Если статус **"Черновик"** - новость видна только вам в админ-панели
- Если статус **"Опубликовано"** - новость доступна публично через API

API для получения новостей:
```bash
GET http://localhost:3000/api/public/news
```

## Возможные ошибки

### ❌ "Ошибка при создании новости"

**Причина:** Таблица `news` не создана в базе данных.

**Решение:** Выполните SQL миграцию из Шага 1.

### ❌ "Заголовок и канал обязательны"

**Причина:** Не заполнены обязательные поля.

**Решение:** Убедитесь, что заполнены "Заголовок" и выбран "Канал".

### ❌ Пустой список каналов в форме

**Причина:** В базе данных нет каналов.

**Решение:** Создайте канал через вкладку "Каналы".

## Структура базы данных

```
channels
├── id (UUID)
├── name (название)
├── slug (ключ)
└── is_active (активен/нет)

news
├── id (UUID)
├── title (заголовок)
├── content (текст)
├── channel_id (ссылка на канал)
├── author_id (ссылка на автора)
├── status (draft/published)
└── published_at (дата публикации)

news_channels
├── news_id (ссылка на новость)
├── channel_id (ссылка на канал)
└── published_at (дата публикации в канале)
```
