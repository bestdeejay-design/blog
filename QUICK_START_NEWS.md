# 📰 Добавление новостей - Краткая инструкция

## ✅ Быстрый старт

### 1. Создайте базу данных
```sql
-- Выполните в Supabase Dashboard → SQL Editor
-- Содержимое файла: supabase/migrations/20260321_create_database_schema.sql
```

### 2. Запустите приложение
```bash
npm run dev
```

### 3. Войдите в систему
- URL: http://localhost:3000/login
- Email: `admin@blog.local`
- Пароль: `Admin@2026Secure!`

### 4. Добавьте канал (если нет)
1. Вкладка **"📺 Каналы"**
2. Нажмите **"➕ Добавить канал"**
3. Название: `Main News`
4. Ключ: `main-news`

### 5. Добавьте новость
1. Вкладка **"📰 Новости"**
2. Нажмите **"➕ Добавить новость"**
3. Заполните форму:
   - Заголовок: `Моя первая новость`
   - Текст: `Текст новости...`
   - Канал: выберите из списка
   - Статус: `Опубликовано` или `Черновик`
4. Нажмите **"✅ Создать новость"**

## 🔍 Проверка

API вернет JSON с созданной новостью:
```json
{
  "success": true,
  "news": {
    "id": "...",
    "title": "Моя первая новость",
    "content": "Текст новости...",
    "status": "published",
    ...
  }
}
```

## 🐛 Исправление ошибок

### Ошибка в консоли / "Ошибка при создании"

**Проблема:** Таблица `news` не существует

**Решение:**
1. Откройте Supabase Dashboard
2. SQL Editor
3. Выполните SQL из файла миграции

### Пустой список каналов

**Проблема:** Нет созданных каналов

**Решение:** Создайте канал во вкладке "Каналы"
